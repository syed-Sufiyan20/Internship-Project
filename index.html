My projects

During my final year project,I had done rain forecast predictor with the help of rain sensor and andruino board. A Rain Sensor IoT system using Arduino is a simple yet effective project designed to detect rainfall and send alerts or control devices accordingly. It typically uses a rain sensor module that detects the presence and intensity of water droplets. The sensor is connected to an Arduino microcontroller, which reads the analog or digital signals and processes them. When rain is detected, the Arduino can trigger actions like sending alerts via Wi-Fi (using ESP8266/NodeMCU), activating alarms, or controlling actuators like closing windows or retracting solar panels. This project is widely used in smart irrigation, weather monitoring, and home automation systems. It’s an excellent application of IoT for real-world environmental monitoring. The Ola webpage interface project focuses on designing a modern, responsive web page inspired by the layout and functionality of the official Ola Cabs platform.
